Generative Adversarial Network
Redes Neurais Generativas (GAN) ^7cW9AxuT

Uma GAN é composta de duas partes:
    1. Gerador: aprende a gerar dados plausíveis. As instâncias geradas se tornam exemplos de treinamento negativos para o discriminador.
    2. Discriminador: aprende a distinguir os dados falsos do gerador de dados reais. O discriminador penaliza o gerador por produzir resultados implausíveis.

- Tanto o gerador quanto o discriminador são redes neurais. A saída do gerador está conectada diretamente à entrada do discriminador.
- Através do backpropagation, a classificação do discriminador fornece um sinal que o gerador usa para atualizar seus pesos. ^n8a5sUyl

Discriminador ^xFKSmhPz

- Classificador que tenta distinguir os dados reais dos dados criados pelo gerador. Pode usar qualquer arquitetura de rede apropriada ao tipo de dados que está classificando.
- Dados de treinamento vem de duas fontes:
    - instâncias de dados reais, usa como exemplos positivos durante o treinamento.
    - instâncias de dados falsos criados pelo gerador, usa como exemplos negativos durante o treinamento.

- Durante o treinamento do discriminador, o gerador não é treinado. Seus pesos permanecem constantes enquanto produz exemplos para o discriminador treinar. ^bRTi5uGd

Gerador ^ubuJx9M0

A parte geradora de um GAN aprende a criar dados falsos incorporando feedback do discriminador. Ele aprende a fazer com que o discriminador classifique sua produção como real.
- O treinamento do gerador requer uma integração mais próxima entre o gerador e o discriminador. A parte do GAN que treina o gerador inclui:
    - entrada aleatória
    - rede geradora, que transforma a entrada aleatória em uma instância de dados
    - rede discriminadora, que classifica os dados gerados
    - saída discriminador
    - perda do gerador, que penaliza o gerador por não enganar o discriminador ^50klH9X4

Entrada Aleatória ^imwVWzsy

Em sua forma mais básica, uma GAN recebe ruído aleatório como sua entrada. O gerador transforma esse ruído em uma saída significativa.
- Ao introduzir ruído, podemos fazer com que a GAN produz uma ampla variedade de dados, amostragem de diferentes lugares na distribuição alvo.
- Experimentos sugerem que a distribuição do ruído não importa muito, então podemos escolher algo que seja fácil de amostrar, como uma distribuição uniforme. ^40qpdyC2

Usando Discriminador para treinar o Gerador ^pcEubuVc

Treinamos o gerador:
    1. Amostra de ruído aleatório
    2. Produza a saída do gerador a partir do ruído amostrado
    3. Obtenha a classificação (real ou falso) para saída do gerador
    4. Calcule a perda da classificação discrimintadora
    5. Retropropagar através do discriminador e do gerador para obter gradientes.
    6. Use gradientes para alterar apenas os pesos do gerador. ^84f4H8Rc

Treinamento GAN ^mPsR6ldD

Como o GAN contém duas redes treinadas separadamente, seu algoritmo de treinamento deve resolver dois problemas:
    - GANs devem lidar com dois tipos diferentes de treinamento.
    - A convergência GAN é difícil de identificar.

Então como treinar a GAN como um todo?
- O treinamento prossegue em períodos alternados:
    1. O discriminador treina por uma ou mais épocas
    2. O gerador treina por uma ou mais épocas
    3. Repita as etapas 1 e 2 para continuar a treinar as redes

- Mantem-se o gerador constante durante a fase de treinamento do discriminador.
- À medida que o treinamento do discriminador tenta descobrir como distinguir dados reais dos falsos, ele tem que aprender a reconhecer falhas do gerador.
- Esse é um problema diferente para um gerador completamente treinado do que para um gerador não treinado que produz saída aleatória.
- Da mesma forma, mantemos o discriminador constante durante a fase de treinamento do gerador.
    - c.c. o gerador estaria tentando atingir um alvo em movimento e nunca poderia convergir. ^pR5WhKzu

Convergência ^fn1illdW

À medida que o gerador melhora com o treinamento, o desempenho do discriminador piora, que não consegue dizer facilmente a diferença entre o real e falso.
- Se o gerador for perfeito, o discriminador terá uma precisão de 50%.
- Essa progressão representa um problema para convergência da GAN como um todo.
    - O feedback discriminador se torna menos significativo ao longo do tempo.
- Se a GAN continuar treinando além do ponto em que o discriminador está dando feedback completamente aleatório, o gerador começa a treinar com feedback indesejado. ^UAVUhhXd

Funções de Perda ^KL9PomJo

Uma GAN tem duas funções de perda: uma para treinamento do gerador e outra para treinamento do discriminador.

- Como elas trabalham juntas para refletir uma medida de distância entre as distribuições de probabilidade?
    - As perdas do gerador e do discriminador derivam de uma única medida de distância entre as distribuições de probabilidade
    - O gerador só pode afetar um termo na medida de distância: o termo que reflete a distribuição dos dados falsos.
    - Então, durante o treinamento do gerador, abandonamos o outro termo, que reflete a distribuição dos dados reais. ^DEDDTFc1

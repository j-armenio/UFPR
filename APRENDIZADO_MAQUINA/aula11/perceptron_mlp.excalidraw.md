- 2014: GANs, Goodfellow, introduziou o conceito de treinamento adversário para modelos generativos.

Multi-Layer Perceptron ^I0adv2Cj

Uma solução para resolver um problema não linearmente separável consiste em adicionar novas características ao vetor.

Considere o problema XOR, que não pode ser resolvido com um perceptron.
- Nesse caso, a característica adicionada x  é a operação AND entre x  e x .
- Ao adicionar essa característica, o problema se torna linearmente separável. ^wrI3vecK

Backpropagation (Descida do Gradiente) ^BGl8UyPH

- Minimizar uma função de erro, a qual define o quanto a saída do modelo difere da saída desejada.
- Uma função bastante utilizada é a soma dos erros dos quadrados.
- BackProp encontra o conjunto de presos que minimiza o erro através da Descida do Gradiente (Gradient Descent). ^kBSGFEbx

Batch
    - usa todo conjunto de dados a cada iteração.
    - calcula o erro para cada exemplo e faz a média para definir a direção do gradiente.

Estocástico
    - para cada exemplo de treinamento, o algoritmo calcula a direção do gradiente.
   
MiniBatch
    - combina os dois anteriores (tamanho típico: 32, 64, 128). ^7UJZe1sV

    - mais utilizada atualmente.

Aspectos Práticos ^CqBwxybZ

Alguns pontos devem ser considerados no uso de MLPs.
- Taxa de aprendizagem (velocidade de convergência)
- Momentum (evitar mínimos locais)
- Shuffle (evitar catastrophic forgetting)
- Normalização (características com média próxima a zero) ^P7olf2A9

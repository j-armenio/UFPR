Aprendizagem Semi-supervisionada ^CUN13Rox

Por que? ^FvfOn5S6

Em várias aplicações temos muitos dados, mas poucos rotulados.
    - Imagens médicas (exames), cyber-segurança (tráfego de rede). ^H96kIZM2

Aprendizagem semi-supervisionada é um ramo da aprendizagem de máquina que combina aprendizagem supervisionada e não supervisionada.
    - Usa dados rotulados e não rotulados.

 ^KrMbVGy5

São especialmente relevantes em situações em que obter uma quantidade suficiente de dados rotulados é proibitivamente difícil ou caro, mas grandes quantidades de dados não rotulados relativamente fáceis de adquirir.
    - Com mais dados, temos fronteiras mais representativas. ^nCVpeLev

Hipóteses ^9bIuR3Bj

- Suavidade (Smoothness): se dois exemplos, x e x' estão próximos no espaço de características, seus rótulos, y e y' são os mesmos.
- Baixa-densidade: a fronteira de decisão não deve passar numa área de alta-densidade no espaço de características, por exemplo, no meio de um cluster. ^JEo8yqoc

- Manifold: dados de alta dimensão estão concentrados em torno de uma variedade (manifold) de dimensão muito menor que o espaço ambiente. Dados não rotulados revelam a geometria do manyfold. ^gRJxXZZx

Métodos ^5xZcyG35

Podem ser divididos em:
- Indutivos
    - Wrapper (self-training, co-training)
    - Pre-processamento (PCA, autoencoder)
    - Generativos (GMM)
- Transdutivos
    - Baseados em grafos (label-propagation) ^FYOr2wUl

Métodos Wrapper ^4b6zAsR2

- Treinar classificadores utilizando dados rotulados.
- Usar esses classificadores para rotular dados (pseudo-rótulos).
- Os classificadores podem ser então re-treinados, usando esses pseudo-rótulos. ^AF0QwUfq

Self-training (self-learning) ^nzaJlc6K

Um classificador supervisionado é treinado em dados rotulados e então é utilizado para classificar dados não rotulados.
- As "melhores" predições são adicionadas na base de aprendizagem.
- Classificador é então re-treinado com os dados originais + dados 
pseudo-rotulados.
- Esse processo continua até que não existam mais dados não rotulados.
    
- Ainda é possível escolher não adicionar dados não rotulados na base se eles foram classificados com baixa probabilidade.
    - quais exemplos serão adicionados a base é de suma importância. ^zyaWwYXD

Co-training ^v25CLIkz

- Extensão do self-training, que usa múltiplos classificadores.
- A cada iteração, os dados rotulados por um classificador são adicionados na base que será utilizada por outro classificador.
    - Importante que classificadores não tenham alta correlação em suas predições.
- Para promover diversidade, classificadores são treinados em diferentes subsets de características (multi-view). ^F4ygNPeq

Pre-processamento Não-Supervisionado ^I1E5yYBi

Uso de dados de forma não supervisionada antes do uso de algoritmos supervisionados.
    - Extração de representações
    - Cluster-than-label
        - Utilizam clustering em todos os dados disponíveis e usam os grupos resultantes para guiar o processo de classificação. ^YRudbhiV

Generativos ^b00WpMwp

Usam dados não rotulados para aprender melhor a distribuição dos dados. ^9Ka8ghUq

- Considere os dados rotulados (X,Y). Assumindo que cada classe tem uma distribuição gaussiana, qual seria a fronteira de decisão? 

- Estimando os parâmetros das distribuições, os dados podem ser classificados como uma GMM.
- GMM pode ser utilizada para gerar dados a partir dos parâmetros estimados, e a partir desses dados gerados, uma nova fronteira de decisão pode ser construida. ^Prhcp4lB

Label Propagation ^4xqkGMhk

Ideia central de tratar os dados como um grafo, onde cada ponto é um nó e as arestas representam a similaridade entre pontos.
- Os rótulos "fluem" dos nós rotulados para os não rotulados através dessas conexões.

Algoritmo:
1. Constrói-se um grafo de similaridade, com k-vizinhos mais próximos.
2. Define-se uma matriz de transição W que pondera a influência entre vizinhos.
3. Os rótulos se propagam iterativamente: cada nó absorve os rótulos de seus vizinhos, ponderados pela similaridade.
4. O processo converge quando os rótulos se estabilizam.

A atualização segue ^w1IoMY13

rótulos iniciais ^lHse7F5r

matriz normalizada ^9u5tFbPI

controla equilíbrio entre a influência dos vizinhos e ancoragem nos rótulos originais ^PEx3ZKPM

Com a alto, o sinal se propaga longe (vizinhos distantes influenciam mais), mas o algoritmo demora mais para convergir e pode ser menos robusto a ruídos no grafo.
- Com a baixo a ancoragem domina e os rótulos viajam pouco, útil quando você confia muito nos poucos pontos rotulados. ^MIM2kYCy

Active Learning ^oVF4wEKO

Como priorizar os dados que devem ser rotulados para ter um alto impacto no treinamento de um modelo supervisionado.
    - Quantidade de dados rotulados é grande
    - Priorizar algumas instâncias pode produzir um modelo melhor. ^ZvFQ9CIY

Algoritmo:
1. Rotular um pequeno conjunto de dados manualmente.
2. Treinar um modelo supervisionado nesse pequeno conjunto.
3. Usar esse modelo para classificar dados não rotulados.
4. Calcular um score para cada dado não rotulado (baseado numa estratégia de prioridade).
5. Adicionar instâncias selecionadas na base de aprendizagem.
6. Repetir passos [2..5] até alcançar um critério de parada. ^pSPvEkvR

Como priorizar? ^kKAdSG6p

Least Confidence ^H3imBtkE

- Método mais simples de priorização.
- Utiliza a probabilidade estimada pelo classificador.
    - ordena em ordem crescente.
- O modelo pede ajuda exatamente nos modelos que está menos seguro.
- Ordem no ex: X3, X2, X1, X4 ^28mlxRkQ

Margin Sampling ^4SJ8dUzO

- Leva em consideração a diferença (margem) entre as duas maiores probabilidades.
- Quanto menor a margem, mais indeciso é o modelo.
- Ordem no ex: X3, X1, X2, X4 ^M9PdIWJc

Exemplo: ^qibxvTKG

Entropy Sampling ^vQUytaDV

- Usa entropia para medir a certeza do modelo.
- Alta entropia, quer dizer que o modelo distribui as probabilidade entre todas as classes.
- Ordem no ex: X3, X1, X2, X4 ^uSRdvdNe

funciona para dados tabulares ou de baixa dimensão, mas falha em imagens brutas: dois pixels vizinhos no espaço RGB não dizem nada sobre semântica. ^uc7fi3Dm

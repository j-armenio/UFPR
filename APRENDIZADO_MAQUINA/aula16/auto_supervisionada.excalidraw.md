Aprendizagem Auto-Supervisionada
Self-Supervised Learning (SSL) ^UDT5cIvg

Toda aprendizagem semi-supervisionada clássica repousa na hipótese de suavidade (pontos próximos no espaço de entrada tendem a ter mesmo rótulo).

Técnicas de SSL baseada em aprendizagem profunda aprendem o próprio espaço de representação onde essa suavidade vale, em vez de assumi-lo a partir de features brutas. ^8zaYu0xo

Constrative Learning ^M4ihbOY0

A ideia central é aprender uma função de embedding f(x) tal que o grafo de similaridade (semelhante a label-prop) implícito faça sentido semântico, sem usar nenhum rótulo.
- Como ele define um "vizinho" sem rótulos?
    - Através de data augmentation: duas versões perturbadas da mesma imagem (crop, flip, jitter de cor) devem ser atraídas no espaço de embedding. Imagens diferentes devem ser repelidas.
        - SimCLR (Simplified Constrative Learning of Visual Representations)
        - MOCO (Momentum Constrast) ^dsRkcadN


Etapa 1: Pre-treinamento (sem rótulos)
Etapa 2: Fine-tunning supervisionado
    - adaptar as representações aprendidas durante o pré-treinamento para uma tarefa específica de classificação.

- Autores mostram que essa estratégia alcança resultados muito bons com apenas 1% dos dados rotulados na ImageNet. ^TfBEkyE9

- O SimCLR precisa de muitos pares negativos no batch para funcionar.
- BYOL e DINO não usam pares negativos.
- DINO se tornou o estado da arte. ^9GYPLckx

DINO (Self-Distillation with No Labels) ^K8EL1eXk

Desenvolvida pela Meta AI que permite treinar Vision Transformers sem necessidade de rótulos humanos.
- Usa uma abordagem de conhecimento destilado.








- Não há um modelo "professor" pré-treinado, o sistema aprende sozinho através da consistência entre diferentes visões de uma mesma imagem.
- Loss Function: o objetivo é fazer com que a saída do Student seja mais próxima possível da saída do Teacher.
    - O Student tenta "adivinhar" a representação global do que o Teacher viu, mesmo só tendo acesso a um detalhe local.  

- Propriedades emergentes:
    - Segmentação automática: Mesmo sem nunca ter visto um rótulo, o mapa de atenção do DINO consegue isolar perfeitamente os objetos do fundo.
    - Representações são tão boas que você pode classificar imagens usando apenas um algoritmo simples de vizinhos mais próximos (k-NN), sem precisar treinar uma camada final de classificação. ^lRPtJlb8

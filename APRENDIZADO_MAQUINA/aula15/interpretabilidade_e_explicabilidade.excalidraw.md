PFI - Permutação Feature Importance ^jWZ0H9oV

Usada para estimar importância em um modelo já treinado, é agnóstica ao modelo, ou seja, pode ser utilizada em qualquer tipo de modelo.
- Diferentes modelos podem apresentar diferentes resultados. ^b64a8SfY

Algoritmo:
1. Treine o modelo normalmente
2. Meça o desempenho base (ex: acurácia, RMSE) em um conjunto de validação/teste.
3. Para cada variável X :
    - Embaralhe (permute) os valores de X  entre as amostras
    - Faça previsões com o modelo sem re-treinar
    - Meça o novo desempenho
4. Importância = queda de desempenho após permutação


- Para métricas de erro (RMSE): quanto maior o aumento, mais importante.
- Para métricas de acurácia: quanto maior a queda, mais importante. ^qkTv51h7

Importance(X ) = Erro      - Erro ^tfJ5D7um

Limitações ^iZ0dW88N

- Quando duas características são correlacionadas, embaralhar uma delas, o modelo ainda pode ter um bom desempenho usando a outra, de forma que, a queda do desempenho é baixa e ela será considerada não importante. Entretanto, ela é importante (compartilha importância com a outra).  
    - Ex: Pressão arterial diastólica e Pressão arterial Sistólica.  ^c7jC1z4v

SHAP Values ^64Hvs5vv

Os valores de Shapley, originados da teria dos jogos, são usados para atribuir importância às contribuições de cada característica (ou variável) em um modelo.
- Eles fornecem uma forma justa de atribuir crédito a cada característica com base no impacto no resultado do modelo.
- Para calcular os valores de Shapley de uma feature específica, são considerados todas as possíveis combinações de características. 
    - Isso é muito caro computacionalmente.
- Uma aproximação que torna o cálculo mais viável é o 
SHAP (SHapley Additive exPlanations) ^JjIBAJB5

Waterfall ^1QlEVtI1

- Mostra os valores SHAP para uma dada observação da base de dados.
- Quanto cada cada característica contribui para a decisão.
- E[f(x)] é o valor médio das predições do modelo para toda base.
- No exemplo, probabilidade de ser maligno é muito baixa. ^QRu2uoVY

- Útil para avaliar os erros do modelo. ^9GXMG3E0

Mean Shap ^aDON3Wj1

- Mostra as características mais importantes.
- O valor médio de todas as observações é usado. ^ymZhQr1h

BeeSwarm ^7ZRAV9MR

- Eixo Y: características em ordem de importância
- Eixo X: valores SHAP
- Mostra como os valores das características estão distribuídos entre as duas classes (esqueda = benigno, direita = maligno). ^gCdjf9M9

LIME - Local Interpretable Model-agnostic Explanations ^LjeElP2d

Explica uma instância específica dos dados.
- Gera variações da entrada (conjunto de dados artificiais) e obtém previsões do modelo para essas variações.
- Com base nas previsões do modelo para os dados perturbados, o LIME ajusta um modelo interpretável e simples (surrogate model), como uma regressão linear que se aproxima das previsões do modelo complexo para a instância específica.
- O modelo simples fornece uma explicação mais fácil de entender sobre como as características da instância influenciam a previsão.  ^yx0lbavi
